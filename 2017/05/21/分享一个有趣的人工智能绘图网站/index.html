分享一个有趣的人工智能绘图网站

发表于 2017-05-21 | 分类于 other

相信不少技术爱好者已经关注到，早些前，谷歌推出了一款线上绘图神器——AutoDraw。它可以一键把你粗糙的涂鸦变成一个精美的图案，而且AutoDraw基本不受设备的限制，不论是在手机、平板、电脑上都能顺畅运作。你只需用鼠标简单的绘出大致形状，AI便会猜你实际上画的是什么，列出可能结果清单，将你的图案变的更专业更好看，后续可以用其他工具修改成想要的画面，并且图片支持下载和分享。先来一个官方宣传视频：AutoDraw官方宣传视频

有没有迫不及待想要动手尝试一下？先别着急，先让我给你介绍一下这个神奇的网站：

首先：打开官网 (https://www.autodraw.com/)

打开网站后会默认给定一个矩形的画图面板，当然可以在左侧菜单栏选择不同样式尺寸的面板：

选中的尺寸也不是一成不变的，选中尺寸后我们还是能够拖动面板右下角的小三角来自定义尺寸：

做好准备工作，接着就了解一下左侧工具栏的一些工具：


Select

Select选择工具，用户可以点击需要选择的图形对象，也可以摁住鼠标拖动选中多个图形同时操作。当图形处于选中状态会出现如下的一个选中框框住，用户可以拖拽小正方形改变图形形状大小，也可以拖拽小圆形控制图形旋转：


AutoDraw

这就是我们今天的主角，它是画笔的一种，只是用AutoDraw画笔画出的图形，后台会根据人工智能识别，”揣摩”用户意图，然后在推荐栏列出一系列精美图案供用户选择，用户可以点击相应的图形替换当前绘制的图形：


Draw

与AutoDraw不同的是Draw就是普通的画笔，用Draw画出来的抽象图形并不会有联想功能，用户可以通过面板左上角的滑块控制画笔笔刷的粗细：


Type

Type是文字工具，选中Type工具并在面板上任意位置点击就可以进行文本输入，并可以通过操作面板的左上角控制字体样式和字体大小：


Shape

Shape则是图形工具，Shape为用户提供了圆形、矩形和等边三角形三种形状的图形，用户只需要选中特定图形，并在面板中摁住并拖动鼠标，就能够绘制相应的图形：


ColorPicker

ColorPicker是颜色选择器，用户只需要将鼠标移动到工具上方，颜色选择列表就会展开，用户可以点击左右移动的按钮选择更多的颜色，美中不足的是颜色值非常有限，要是Google官方能够优化一下，可以根据给定的RGB值来选择颜色就更完美了:


Fill

之前绘制出来的图形都是描边图形，而且颜色非常单一。为了让我们的作品显得更加鲜艳动人，我们就要来给我们的作品上色，就像小时候我们买来图画本用水彩笔涂上喜欢的颜色一样，这时候就需要用到Fill工具了，Fill既可以填充颜色，也可以改变描边颜色，填充色为ColorPicker选中的颜色：


Zoom

Zoom缩放工具，控制面板的缩放：


Undo

Undo撤销工具，撤销上一次操作，对于PC用户这个工具貌似有点鸡肋，快捷键（ctrl+z）更加方便。


Delete

Delete删除工具，一般时候这个工具都是不可操作状态，只有当前有操作对象或是选中对象是才会显示可操作状态，删除当前选中或者操作对象，对于PC用户这个工具貌似也很鸡肋，快捷键（delete）不要太方便。

到这里AutoDraw的全部功能就介绍完啦 ！

嗯，这个… 还有一个事情需要温馨提示一下，如果你的设备没有翻墙的话，这篇文章你基本上是也就是看看，没什么好激动的，因为根本没你什么事情，AutoDraw这个网站虽然能够打开，但是需要翻墙才能体验智能联想的效果。（PS：说了要先读完文章在试吧，那些没读完文章就开始尝试的，估计还傻傻的以为网速不行，哈哈 ~）关于这个问题，还是希望Google官方能尽快能够解决，让我们大陆用户能体验这个人工智能领域的有趣的网站吧！
